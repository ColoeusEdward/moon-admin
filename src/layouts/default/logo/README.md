## 头部logo组件

### 参数

| 名称 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| logoStyle | `object` |  | 头部logo组件样式对象 |
| logoImage | `string` |  | 头部logo图片 |
| logoName | `string` |  | 头部logo名称 |

### 样式参数

| 名称 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| height | `string \| number` | `70px` | 头部logo组件高度 |
| width | `string \| number` | `270px` | 头部logo组件宽度 |
